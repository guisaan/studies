Python é uma linguagem de programação generalista, multifunções;
Agora vamos aprender os fudamentos da linguagem

Sempre que desenvolvemos um codigo, ele é baseado em algum algoritmo. Como uma receita de bolo. No caso do código, será o que desejamos executar das estapas.

1. Entrada de Dados - os possiveis dados que devem ser lidos antes da rotina programada
2. Processamento de Dados - executar as ações necessária para chegar na saída.
3. Saída de dados - resultado final da programação.

(input) - entrada de dados
(print) - saída de dados

Quando estamos criando nossa códigos, ao pedir informações precisamos guardar essas infos. E fazemos isso através das variáveis, que são "fragmentos dentro da memória" que você tem disponível.
E para resgatar essa informação você utiliza a variável.

Outro recurso são os comentários, utilizando o símbolo # no inicio de um texto para criar um comentario.
